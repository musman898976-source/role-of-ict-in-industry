# Industrial Cybersecurity Protocol (ICS/SCADA)

Industry ko cyber attacks se bachane ke liye ye **Professional Checklist** follow karein:

- [ ] **Network Segmentation:** Separate the Corporate Office network from the Factory Floor network.
- [ ] **Physical Security:** Lock all USB ports on PLC machines to prevent unauthorized malware.
- [ ] **Encryption:** Use AES-256 encryption for all wireless sensor data.
- [ ] **Multi-Factor Authentication (MFA):** Ensure all remote access to the factory requires MFA.
- [ ] **Regular Backups:** Weekly offline backups of PLC logic and ERP databases.
