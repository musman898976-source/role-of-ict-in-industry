# Industrial ICT Framework (Standard: ISO/IEC 27001 & ISA-95)

To make an industry "Smart," you must follow the **ISA-95 Hierarchy**:

| Level | Function | Technology Involved |
| :--- | :--- | :--- |
| **Level 4** | Business Planning & Logistics | ERP (Odoo, SAP, ERPNext) |
| **Level 3** | Manufacturing Operations (MES) | Batch Control, Quality Tracking |
| **Level 2** | Monitoring & Supervising | SCADA Systems, HMI |
| **Level 1** | Sensing & Manipulation | PLCs, Sensors, Actuators |
| **Level 0** | The Physical Process | Motors, Pumps, Assembly Lines |

### Implementation Steps:
1. **Audit:** Assess current manual processes.
2. **Infrastructure:** Deploy Industrial Ethernet and Wi-Fi 6.
3. **Data Collection:** Install IoT Gateways to fetch data from Level 0.
4. **Analysis:** Use Big Data tools to predict machine failure (Predictive Maintenance).
